1.	Scope: Only apply style rules to description, skills name and keywords. Do not modify codes, categories, levels, evidence or confidence scores.

2.	Core style requirements the model must meet: The handbook expectations translate into the following operational rules.
a.	Plain language and concision
•	Use common, short words.
•	Remove unnecessary qualifiers and abstractions.
•	Avoid bureaucratic phrasing.
•	Target ~15 words per sentence, maximum 25.
Proposed instruction:
“Rewrite in plain language. Be concise. Remove unnecessary words.”
b.	Active voice by default
•	Prefer explicit actors.
•	Avoid agentless passive constructions unless the actor is genuinely unknown.
Detect and rewrite patterns like:
“X is undertaken / is conducted / is processed”
c.	Sentence length control
•	Split long or compound sentences.
•	Lists are acceptable if they improve clarity.
Post processing rule:
If a sentence exceeds 25 words, split or re prompt.
d.	Numbers formatting
Apply the handbook rules consistently:
•	Use words for zero and one.
•	Use numerals for 2 and above.
•	Use commas in thousands.
•	Express large rounded numbers as 1 million, 4.5 million.
•	Do not start sentences with numerals.
•	Ensure comparisons use numerals on both sides.
e.	Keywords list normalisation
•	Use parallel structure.
•	Prefer noun phrases.
•	No mixing verbs, sentences and nouns.
f.	Australian English and accessibility
•	Australian spelling only.
•	Avoid unexplained jargon or acronyms.
•	Use inclusive, neutral language.
